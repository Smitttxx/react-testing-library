# Paycast

### Prerequisites

- Yarn

- [Cognito user account](https://shieldpay.atlassian.net/wiki/spaces/DEV/pages/2806120449/Amazon+Cognito+user+account)

### Local environment variables

You will need to create a file named `.env` in the Paycast root directory.

```
$ cp .env.example .env
```

#### Installation & Development

```
$ yarn install
$ yarn dev
```
